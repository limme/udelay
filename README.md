# urelay
usb relay by uart 
```
usb_relay.sh -h
usag:
	-u	 power up
	-d	 power down
	-s	 device default /dev/ttyUSB0
	-t n	 delay n second,default 0.5s
	-ud	 power up then power down
	-du	 power down then power up
 	-dut 2	 power down and delay 2s then power up
```
